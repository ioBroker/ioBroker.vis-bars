Haupteinstellüngen

Filter key

Gibt einen Schlüsselwort ein, über welches das Widget mit dem Filter ausgewertet wird. Allerdings für Filter Bar gibt es keinen Sinn einen FilterKey einzustellen, weil so kann es sich selbst ausfiltern.

Zeige in Views

Gerade für Bar-Widgets ist das relevant ein Bar-Widget in mehr als einem View zu zeigen. Hier kann man einstellen in welchen Views es noch angezeigt werden soll.
Es erscheint an der gleichen Stelle in anderem View.

Thema

Hier wird das Thema für ganzen Bar eingestellt. Die Buttons selbst sind nicht beeinflusst.
Aktuell sind drei Themen unterstützt: Dark, Blue, Red
Man kann aber jedes beliebigen Theme eintragen, sogar gar nichts eintragen, dann wird das Bar transparent.

Filter bar


Dieses Widget kann sehr schnell einen Set von Filter-Knopfe erstellen.
Mit dem Eigenschaft "FilterKey" jedes Widget kann zu einer Filter-Gruppe zugeordnet werden.
Filter Bar kann diese Gruppen auswählen. Falls keine Gruppe markiert ist, dann werden alle Gruppen angezeigt.

Haupteinstellüngen

  • Theme
  • Nur eine auswählbar
    Ob nur einen Filter gleichzeitig aktiv werden kann oder mehrere.
  • Initial Filter
    Dieses Filter wird beim ersten Anzeigen automatisch ausgewählt
